DOM Manipuláció

Metódusok

Megragadja az első megadott elemetdocument.querySelector({HTML Element});

Megragadja az összes megadott elemetdocument.querySelectorAll({HTML Element});

Új attribútum értékek hozzáfűzése elemekhezadd();

Attribútum értékeket lehet törölni elemkbőlremove();

Ki-be lehet kapcsolni az elemekhez tartozó attribútumokattoggle();

Property-k

A megragadott elem tartalma innerHTML

A megragadott elem és tartalma outerHTML

Az adott elemhez tartozó id id

Az adott az elemhez tartozó osztályok nevei className

Az adott elemre feliratott osztályokból készít egy listát amiből lehet törölni és hozzáadni is classList

Új DOM elem létrehozása

  1. Készítsd el az elemet var {elementVariable} = document.createElement({tagName}[, options])

  2. Készítsd el a tartalmat amit hozzá akarsz majd fűzni var {contentVariable} = document.createTextNode("{String}}");

  3. Add az elemhez a tartalmat {elementVariable}.appendChild({contentVariable})

  4. Add az elkészült elmet a DOM Treehez document.querySelector({HTML Element}).appendChild({elementVariable})

Esemény Kezelés

Az esemény kezelés lépései:

  1. Azonosíts be egy eseményt
  2. Rendelj egy viselkedést a kiválasztott DOM elemhez
  3. Futtasd a függvényt ha bekövetkezik

Esemény kezelő létrehozása

                    {elementName}.addEventListener("{eventName}",{eventHandlerMethod},{hasEventBubbeling})

Argumentumok átadása esemény kezelőnek

                    function {eventHandlerMethodName}(e,current) {
                        {Logic}
                    }

                    {elementName}.addEventListener("{eventName}", function(e){
                        {eventHandlerMethod}(e, this)
                    }, {hasEventBubbeling})
