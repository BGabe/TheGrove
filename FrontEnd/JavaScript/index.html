Mire érdemes figyelni Javascript használatakor:

  • Használd a Console-t a játszóteredként hogy ha kiakarsz próbálni egy kódot
  • A sctipt tag elhelyezése számít pl: Ha a HTML Head részében helyezzük el akkor a scritp nem tudja manipulálni a DOM-ot hiszen még le sem generálódott
  • A Javascript Case Sensitive
  • A ; jel használata nem kötelező de ajánlott a kódok végén
  • Escape Karakter: \
  • Egysoros komment: // {comment}
  • Többsoros komment: /* {comment} /*
  • NaN: Not a Number

Változók

  • Konstans értékekhez: const {variableName}; (Konvenció: Csupa nagybetűvel írd a változó nevét)
  • Globális vagy lokális változókhoz: var {variableName}; (Konvenció: camelCase)
  • Blokk szintű változókhoz: let {variableName}; (Konvenció: camelCase)

Adat Típusok

Null: Nemlétező érték (Nézz utána !)

Number: Számoks

String: Szövegek és szimbólumok megadásuk: "{string}" vagy '{string}'

Boolean: Logikai

Undefined: A változó létre lett hozva de nincs értéke

Array: {var/let/const} {variableName} = [{itemOne},{itemTwo}]

    Fontosabb Metódusai:
  • reverse(): Megfordítja a tömb elemeinek sorrendjét
  • shift(): Törli a tömb első elemét
  • unshift({newValue}): Beszúrja a megadott elemet a tömb elejére
  • pop(): Törli a tömb utolsó elemét
  • push({newValue}): Beszúrja a megadott elemet a tömb végére
  • splice({startIndex},{numberOfRemoves}): Megadott pozíció után megadott számú elemet töröl
  • slice(): Másolatot készít a tömbről
  • indexOf({searchTarget},{startingIndex}): A megadott elemet a megadott pozíciótól kezdve keresi
  • join({separator}): Fogja a tömb összes elemét, stringé alakítja és a megadott elválasztással el tördeli a stringet

Map: Kulcs-érték pár alapon működik, ahol a beszúrási sorrend számít

Sets: Értékek gyűjteménye {var/let/const} {variableName} = new Set();

  • Vegyes típusokat is tartalmazhat
  • Minden értéknek egyedinek kell lennie

Symbols: Egyedi azonosítokhoz lehet használni: const {variableName} = Symbol();

Operátorok

Type Operator: Kideríthető vele egy változó pillanatnyi típusa typeof {variableName}

Assigment Operator: Értékadásra használható {variableName} = {value}

Arithmetic Operators: Aritmetikai műveletekre használhatóak + - * /

    Shorthandek:
  • +=
  • -=
  • *=
  • /=

Unary Operator: Értékek növelése/csökkentése inkrementálisan {variableName}++/--

Lose Equality Operator: Ha két változót értékek alapján akarok összehasonlítani {variableName} == {variableName}

Strict Equality Operator: Ha két változó értékek és típus alapján akarok összehasonlítani {variableName} === {variableName}

Comparator Operator: Össze hasonlításra használhatóak < > >= <=

Lose Not Equal Operator: Ha két változót értékek alapján akarok összehasonlítani {variableName} != {variableName}

Strict Not Equal Operator: Ha két változó értékek és típus alapján akarok összehasonlítani {variableName} !== {variableName}

Logical Operators:

  • AND: És &&
  • OR: Vagy ||
  • XOR: Kizáró vagy ({variableName} == {variableName} || {variableName} == {variableName}) && ({variableName} == {variableName} != {variableName} == {variableName})

Treanry Operator: Egyszerű döntésekhez {variableName} {logicalOperator} {variableName} ? {trueStatemant} : {falseStatemant}

Spread Operator: Megadott tömb elemeit metódus paraméter alakítja ...{arrayName}

Ciklusok

Ciklus megállítása: break;

Iteráció léptetése befejezés előtt: continue;

For of Ciklus

                        for (let {loopVariable} of {variableName}){
                            {Statemant}
                        }

Hibakezelés

Ha tesztelés miatt ki akarunk iratni vlaamit consolba: console.info();

Ha hibát akarunk dobni: console.error();
