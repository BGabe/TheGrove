Függvények

Named functions

Akkor előnyös ha többször is akarjuk használni vagy nagy mennyiségű kódot tartalmaz 

                        function {functionName}({parameter}) {
                            {Logic}
                        }

Anonymous functions

                        (var/let/const) {variableName} = function({parameter}) {
                            {Logic}
                        }

Arrow functions

  • Egy paraméter és egy soros kifejezés: 
                                    var {variableName} = {parameterName} => {Logic}
  • Paraméternélkül vagy többmint egy paraméteres több soros kifejezés 
                                    var {variableName} = ({parameterName,parameterName}) => {
                                    {Logic}
                                    return {value}
                                }

Generators

Olyan metódusok amiknek a futását meglehet állítani és később folytatni lehet 

                        function {functionName}() {
                            {Logic}
                            yield; Ezzel megállítjuk a futását és lezárjuk a szakaszt
                            {Logic}
                        }
                        {function}.next() A függvény következő szakaszának indítása

Osztályok

Nem lehet őket felül írni

Osztály létrehozása

                        class {className} {
                            consturctor({propertyName},{anotherPropertyName}) {
                                this.{property} = {property};
                                this.{anotherProperty} = {anotherProperty};
                            }
                            {methodName}({parameter}) {
                                {Logic}
                            }
                        }

Példányosítás

                        var {variableName} = new {class};

Öröklődés

Származtatás: class {childClass} extends {parentClass}

Ős osztály konstruktorának a használata: super({arguments});

Ős osztály metódusainak a hívása: super.parentClassFunction({arguments});

Setter

                    set {methodName}({parameterName}) {
                        this.property = property;
                    }

Getter

                    get {methodName}() {
                        return this.property;
                    }

Statikus Metódusok

static {methodName}

Prototype

Mixins

Több osztály össze komponálása 

                    let mixin = {
                        {methodName}() {
                            {Logic}
                        }
                    }

                    let {methodNameMixin} = {
                        __proto__: mixin,

                        {methodName}() {
                            super.{methodName}();
                        }
                    }

                    Object.assign(this.methodNameMixin);

Objektumok

Létrehozás: {variableType} {objectName} = new {object}();

Dot notation: {object}.{property/method};

Bracket notation: Ha property vagy metódus nevében olyan karakter van ami problémát okozhat {object}["{property/method}"];

Propertyk

Property létrehozása: {object}.{propertyName}

Shorthand: 

                    {variableType} {objectName} = {
                        {propertyName}: {value};
                        {propertyName}: {value};
                        {methodName}({parameter}) {
                            {Logic}
                        }
                    }

Konstruktorok

Konstruktor létrehozása: 

                    {objectName}({propertyName},{propertyName}) {
                        this.{property} = property;
                        this.{property} = property;
                        {methodName}({parameter}) {
                            {Logic}
                        }
                    }
