HTML Forms

  • Form: <form action="{URL}" method="{HTTP Method}">Ezzel kezdünk minden form-ot</form>
  • Label: <label for="{Input id}" Címke az Input mezőkhöz></label>
    • Ha az input taget közé rakjuk autómatikusan hozzá rendelődik a label
  • Input: <input name="{Name}" id="{Label For}" type="(text) (email) (password) (search) (tel) (color) (file) (date) (checkbox)" {required} placeholder="{Placeholder value}" value = "{Default Value}">
  • Button: <button type="{Button Type}">Gomb elhelyezése</button>
  • Textarea: <textarea>Nagyobb mennyiségű szöveg beviteléhez</textarea>
  • Select List: <select>Lista</select>
    • Option: <option>Válasz lehetőség</option>
  • Fieldset: <fieldset>Ha Label-eket és Chechkboxokat akarunk össze csoportosítani</fieldset>
    • Legend: <legend>Címke field setekhez</legend>
