Mire érdemes figyelni HTML oldal tervezésekor:

  • Amit meglehet oldani HTML-el azt old meg azzal (Mivel ez a leghibatűrőbb nyelv a CSS és JavaScript-hez képest)
  • A HTML feladata hogy kapcsolatot teremtsen ember és gép között, tehát tervezéskor a szempont egyaránt emberek és gépek számára jól értelmezhető oldal kialakítása
  • Inspirációért érdemes más a projektedhez hasonló oldalak felépítését elemezni
  • Az URL-ek szempontjából fontos a projekt mappaszerkezetének megfelelő felépítése
  • A JavaScript file-ok linkjeit érdemes az oldal végére rakni a gyorsabb betöltés érdekében
