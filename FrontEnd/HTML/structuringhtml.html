Boilerplate

                       
                        <DOCTYPE! html> Melyik HTML verziót használjuk
                        <html> Ezzel jelöljük a HTML file-unk kezdetét és végét
                            <head> Ide ketül minden olyan információ az oldalunkról amit a Böngészőnek tudnia kell
                                <meta> Metadatok ide kerülnek (Karakterkódolás) (Oldal leírása)
                                <link href="{URL}" rel="{Kapcsolat típusa}"> Kapcsolódó file-okat itt lehet megadni (Stíluslapok) (Ikonok)
                                <title></title> Weboldalunk hivatalos címe
                            </head>
                            <body> Minden olyan tartalom amely megjelenik a Weboldalunkon
                            </body>
                        </html>

Semantical Elements

  • <header> </header>Fejléc (Logó) (Navigáció) (Címek) (Szalagcímek)
  • <main> </main>Itt kell az oldal fő tartalmát elhelyezni [Ezt csak egyszer szokás használni oldalanként]
  • <footer> </footer> Oldalak vagy Bejegyzések végén érdemes használni apróbetűs részekhez pl: (Copyright) (Referencia Linkek)
  • <nav> </nav> Navigációs részhez
  • <article> </article>Olyan elemeket teszünk bele melyek egy tartalom egységet képeznek pl: (Bejegyzések) (Posztok) (Kártyák)
  • <section> </section>A tartalom részekre bontásához használható pl: (Fejezetek)
  • <aside> </aside> Mellékes elemekhez pl: (Hirdetések) (Oldalsáv) (Érdekességek)
  • <figure> </figure>Grafikus elemeket lehet elhelyezni benne
    • <figcaption></figcaption> A Grafikus elemhez tartozó leírás köré kell tenni

Non Semantical Elelements

  • <div></div> Ha nincs olyan tag ami passzolna egy Block-line elemhez
  • <span></span> Ha nincs olyan tag ami passzolna egy In-line elemhez
