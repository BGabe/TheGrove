Headlines

  • <h1>Legfontosabb elem</h1>
  • <h2>Nagyon fontos elemek</h2>
  • <h3>Fontos elem</h3>
  • <h4>Kicsit fontos elem</h4>
  • <h5>Kisjelentősségel bíró elem</h5>
  • <h6>Legkevésbé fontos elem</h6>

Text

Paragrafs: <p>Egyszerű szöveget teszünk bele</p>

Code Snippet: <code>Program kódok köré lehet tenni</code>

Pre-processed: <pre>Ha szeretnénk hogy az adott részen az intendációkat figyelembe vegye a böngésző [Code Snippets-eknél hasznos]</pre>

Small Print: <small>Apróbetűs részekhez pl: (Copyright) (Jogiinformációk)<small>

Date and Time: <time datetime="{yyyy-mm-dd}/{hh-mm-ss}">Ha konkrét időpontot vagy idő intervallumot akarunk megadni</time>

Link: <a href="{URL}">Ha linké akarunk alakítani egy részt</a>

Charachters

  • Italic: <i>Ha vizuálisan megakarunk különböztetni egy részt</i>
  • Emphasized: <em>Ha verbálisan is megakarunk különböztetni egy részt (Screen Readerek)</em>
  • Bold: <b>Ha vizuálisan kiakarsz emelni egy részt (Inkább ezt használd mint egy divet)</b>
  • Strong: <strong>Ha verbálisan is kiakarsz emelni egy részt</strong>
  • Lower Index: <sub>Ha alsó indexé akarunk alakítani egy részt<sub>
  • Upper Index: <sup>Ha felső indexé akarunk alakítani egy részt</sup>

Quotes

  • Cite: <cite>A szöveget idéző elem köré tesszük</cite>
  • Blockquote: <blockquote>Több soros idézetek köré tesszük [Block-line]</blockquote>
  • Quotation Mark: <q>Idézet rész köré tesszük [In-line]</q>

Lists

  • Unordered List: <ul>Hogyha a listába rendezett elemek sorrendje nem számít</ul>
  • Ordered List: <ol>Hogyha a listába rendezett elemek sorrendje számít</ol>
  • Definition List: <dl>Hogyha olyan elemeket rendezünk egy listába amikhez leírásokat akarunk társítani [Olyan mint egy C# Dictionary]
    • <dt>Egy állítás</dt>
      • <dd>Leírás</dd>
    </dl>

Tables

Table: <table>Ha létre akarunk hozni egy táblázatot</table>

Table Row: <tr>Sort ad a táblázatunkhoz</tr>

Table Header: <th>Ha oszlop fejlécet akarunk</th>

Table Data: <td>Ha cellát akarunk létrehozni egy sorban</td>
