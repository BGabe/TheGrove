Floatolás

Float: float: {left/right};

Display Flow visszaállítása: clear: {left/right/both};

Ha túlszalad a Folatolt elem a konténerén: overflow: {hidden/auto/scroll}

Clearfix Hack:Ezt akkor érdemes használni ha nem tudunk hová clear propertyt beszúrni

Box Model Fix:

  1. Padding és Border property-k értékeinek hozzáadása a Width és Height property-k értékeihez: box-sizing: content-box;
  2. Width és Height property-k értékeiből kerül levonásra a Border és a Padding property-k értéke: box-sizing: border-box;

Pozícionálás

Pozíció Típusa:

  • Független: position: static
  • Jelenlegi pozícióhoz képest relatív: position: relative
  • Szülőelemhez képest ralatív: position: absolute
  • Viewporthoz képest relatív: position: fixed
  • Viewporthoz és a szülőelemhez képest relatív: position: sticky

Pózíció beállítása:

  • top: {Value};
  • right: {Value};
  • bottom: {Value};
  • left: {Value};

Stackelés

Ha az alapvető Stacking Ordert szeretnénk befolyásolni: z-index: {Number};

Stacking Order:

  1. Root Element
  2. Block-level elements
  3. Floated Elements
  4. In-line elements
  5. Positioned elements

Flexbox

Egydimenziós elrendezésekhez (Horizontális vagy Vertikális)

Flex Container: A szülőelem display property-jét kell beállítani:

  • {felx} Ha Block-line viselkedést akarunk
  • {inline-flex} Ha in-line viselkedést akarunk

Fő tengely irányának megadása: flex-direction: {row/row-reverse/column/column-reverse};

Ha az elemek nemférnek el egymás mellett: flex-wrap:

  • {nowrap} Az elemek mérete dinamikusan változik hogy mindig elférjenek egymás mellett
  • {wrap} Ha nincs elég hely a sor eltörik

Shorthand: flex-flow: {flex-direction} {flex-wrap};

Flex itemek kezdeti méretének beállítása: flex-basis: {LengthValue};

Flex itemek fölösleges hely kitöltése: flex-grow: {Number}

Flex itemek méretcsökkenése hely hiányában: flex-shrink: {Number}

Shorthand: flex: {growValue} {shrinkValue} {basisValue};

Flex itemek főtengelyen való igazítása: justify-content:

  • {start} Flownak megfelelő elhelyezkedés
  • {center} Főtengelyen középre igazítás

Flex itemek kereszt tengelyen való igazítása: align-items

Grid

Grid Container: A szülőelem display property-jét kell beállítani:

  • {grid} Ha Block-line viselkedést akarunk
  • {inline-grid} Ha in-line viselkedést akarunk

Ha az egész Grid Konténert igazítani akarjuk: justify-content: {position};

Grid Items: Csak a szülő elemek közvetlen leszármazottai

  • Elem elhelyezése oszlopban: grid-column: {Start Number/CustomName} / {End Number/CustomName};
  • Elem elhelyezése sorban: grid-row: {Start Number/CustomName} / {End Number/CustomName};
  • Shorthand: Ha a {CustomName}-et úgy adjuk meg hogy oda tesszük hogy -Start vagy -End akkor ha értékül adjuk őket és elhagyjuk az előbbi suffixeket azzal összevonunk
  • Elem elhelyezése Grid Area-val: grid-area: {AreaName};

Sorok és oszlopok egyértelmű megadása:

  • grid-template-column: {firstColumnCustomName} {firstColumnLength}...
  • grid-template-row: {firstRowCustomName} {firstRowLength}...

Autómatikusan generált sorok és oszlopok szabályzása

  • grid-auto-column: {columnLength}
  • grid-auto-row: {rowLength}

Grid Area: Előre el nevezzük a Gridünk területeit majd azokat használjuk a tartalom elhelyezésére

    grid-template-areas:
  • "{AreaName} {AreaName} {AreaName}"
  • "{AreaName} {AreaName} {AreaName}";
  • Ha egy {AreaName} helyére . teszek akkor az a cella üres marad

Grid Gap: Sorok és oszlopok között lévő üres tér: grid-gap: {RowLengthValue} {ColumnLengthValue};

Mértékegységek

Absolute: Fix egységek, szülőelem nem befolyásolja az értékét

  • Pixel: {Number}px Ez az ideális választás ha Absolute mértékegységre van szükség, egész számként kell megadni

Relative: Relatív egységek az Ős és a szülőosztály tulajdonságaitól függ az értékük

  • em: 1em = az örökölt mérettel, lehet törtként is alkalmazni
  • rem: Root em, 1rem = az ős elmemtől örökölt mérettel

Fraction Unit: Az adott konténer elemben elérhető terület megadott részének felel meg

Mértékegység Metódusok:

  • repeat({Number of Repeats}, {LengthUnit})
  • minmax({MinLengthValue}, {MaxLengthValue})
