Font ellenőrző

Typeface kiválsztásánál két szempont van:

  1. Legyen könnyen olvasható
  2. Stílusa egyezzen meg üzenetünk tónusával

Gyakori Typefacek:

  • Scripts: Stílusa a kézzel írott szövegekhez hasonlít, elegáns és főleg meghívókhoz illik, kisméretben nehezen olvasható (cursive fonts)
  • Decorative: Fejlécekhez vagy érdekességekhez illik (fantasy fonts)
  • Monospace: Program kódhoz használják (monospace fonts)
  • Serif: Tradicionális, formális dokumentumokhoz illik (serif fonts)
  • Sans-Serif: Modern, általánosan használható (sans-serif fonts)

Font Stackek:

  • font-family: ({primaryFont},{secondaryFont},{fallBackFont})
  • 2-3 fontnál többet nem érdemes megadni

Font Face-ek: Így csatolhatunk Font fileokat az oldalunkhoz 

                        @font-face {
                            font-family: "{customFontName}"
                            src: url({URL})
                        }

Font Weight Nevezéktan:

  • 100 - Thin
  • 200 - Extra Light
  • 300 - Light
  • 400 - Normal
  • 500 - Medium
  • 600 - Semi Bold
  • 700 - Bold
  • 800 - Extra Bold
  • 900 - Black

Font Property Sorthand: font: {paramtersInOrder}

  1. font-style (Opcionális)
  2. font-variant (Opcionális)
  3. font-weight (Opcionális)
  4. font-size
  5. line-height (Opcionális)
  6. font-family

Text Decoration Property Sorthand: text-decoration: {paramtersInOrder}

  1. text-decoration-line
  2. text-decoration-color
  3. text-decoration-style
