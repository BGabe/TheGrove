Animációk

Keyframe: {Number}% arra van hogy az animáció hosszának a megadott százalékát átlépve érvénybe lépjen a változás 

                        @keyframes {animationName} {
                            {Number}% {transform: {method}({Value});}
                            {Number}% {transform: {method}({Value});}
                            ...
                        }

Animáció Tulajdonságok:

  • animation-name: {animationName};
  • animation-duration: {time};
  • animation-timing-function: {timeFunction};
  • animation-iteration-count: {Number/infinite}
  • animation-delay: {time};
  • animation-fill-mode:
    • none
    • backwards: Az animáció végén vissza ugrik az elem a kezdőpontba
    • forward: Az animáció végén megtartja az elem a helyét
    • both: Az elem "vissza ugrik" a kiinduló pontra de vizuálisan nem történik változás
  • animation-direction:
    • normal: Elejétől a végéig játszódik le az animáció
    • reverse: Végétől az elejéig játsza le az animációt
    • alternate: Minden egyes iteráció alkalmával változik hogy elejétől végéig vagy végétől az elejéig játszódik le az animáció
    • alternate-reverse: Minden egyes iteráció alkalmával változik hogy végétől az elejéig vagy elejétől a végéig játszódik le az animáció
  • animation-play-state: {running/paused}

Shorthand: animation: {AnimationName} {AnimationDuration} {AnimationDelay} {TimingFunction} {FillMode}, {SecondAnimationName} {SecondAnimationDuration} {SecondAnimationDelay} {SecondTimingFunction} {SecondFillMode};

Átmenetek

transition: {Property} {duration} {timeFunction} {Delay};

Asszimetrikus Transzformáció: Ha ki indulási Selectrohoz a visszatérési átmenetet a végső Selectorhoz pedig a ki indulási átmenetet teszed.

Timing Functions:

  • ease: Lassan kezd, majd gyorsul de lassuló tempóval
  • linear: Egyenletesen változik a sebesség
  • ease-in: Lassan indul majd végig gyorsul
  • ease-out: Gyorsan indul majd végig lassul
  • ease-in-out: Lassan kezd, majd gyorsul, majd megint lassul
  • Egyedi: cubic-bezier({x1},{y1},{x2},{y2}) vagy Cubic Brezier Tool

Sprite Animációkhoz Step Graph:

  • step-start
  • step-end
  • steps(n, start)
  • steps(n, end)

Transzformáció

Az elváltoztatás mindig az elem abszolút helyzetéhez képest történik ezért fontos a változások sorrendje!

2D Transzformációk

  • Eltolás: translate({xValue},{yValue})
  • Forgatás: rotate({xValue},{yValue})
  • Méretváltoztatás:scale({xValue},{yValue})
  • Ferdén Eltolás: skew({xValue},{yValue})
  • Az elváltozás pontjának megadása: transform-origin({position},{direction})
